firefly-1.2发布 · 2012-02-16

firefly好久没有发布新版本了，距1.1版发布已经有大半年的时间了。 1.2版的firefly有了较大的更新，主要是增加了模板引擎和应用服务器，并且和IOC/MVC框架进行了无缝整合。 这样web开发的技术栈基本就做全了，firefly也从一个单纯的IOC/MVC框架变成了一个一站式应用程序框架。

虽然firefly功能越来越多，但是追求简单好用、高性能的宗旨是不变的，正如firefly这个名字一样，保持轻盈，化繁为简。

下个版本准备在common模块中增加一个ConcurrentLRUHashMap以及json parser。

更新日志：

  1. 增加了模板引擎、嵌入式应用服务器两大模块
  2. MVC框架小部分功能改进
  3. common增加了一些工具类
  4. bug修复

By Pengpeng

What?

Firefly是一个高性能一站式Web框架。 涵盖了web开发的主要技术栈。 包含Template engine, IOC、MVC framework, HTTP Server, Common tools, Log, Json parser等模块。

Feature

  • 使用简单，基于约定优于配置的思想
  • 精简、轻量、无侵入
  • 高性能，详情见性能测试
